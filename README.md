# 🧠 Mind Garden

A simple yet engaging self-help web app that helps you plant, track, and grow positive thoughts like a digital garden 🌱.

## 🌟 Features (Planned)
- ✍️ Daily thought journaling
- 🌿 Grow positive habits through daily watering
- 🧘‍♂️ Mood tracker with animations
- 📊 Streaks and mental growth statistics
- 🔔 Gentle motivational reminders
- 📖 Inspiring quotes and reflections
- 🎨 Beautiful UI with ambient visuals (like Lottie animations)

## 🚀 Tech Stack
- HTML / CSS / JavaScript
- LocalStorage for now (future: Firebase or Supabase)
- Lottie for animations
- Optional: PWA support
